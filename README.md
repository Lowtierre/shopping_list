Shopping List (Client-side)

A simple shopping list app you can host on GitHub Pages.

## Features
- Essentials catalog (food, cleaning, etc.) to quickly add items
- Effective list (add custom items, remove items)
- Download list as `.txt`
- Saves in browser via LocalStorage

## Run locally
Just open `index.html` in your browser.

## Deploy on GitHub Pages
1. Create a new GitHub repo (e.g. `shopping-list`)
2. Upload `index.html`, `styles.css`, `app.js`, `README.md`
3. Go to **Settings â†’ Pages**
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Save. Your site will be published at the GitHub Pages URL.
